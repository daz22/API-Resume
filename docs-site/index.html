Architecture

Client → Cloudflare → Linode → Nginx → FastAPI

Security controls

  • FastAPI bound to 127.0.0.1 only
  • Nginx reverse proxy on 443
  • Cloudflare proxy enabled
  • Origin certificate installed
  • Cloudflare SSL mode Full strict
  • UFW restricts 80 and 443 to Cloudflare IP ranges only
  • Nginx rate limiting enabled
  • HSTS enabled

Build steps

  • Configured Cloudflare DNS and Origin TLS
